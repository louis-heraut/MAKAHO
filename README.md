# MAKAHO

[<img src="https://github.com/super-lou/MAKAHO/blob/0fad3c354954ebc2d8605a2ec9dd3d4f11a36920/www/screen.png" width="600">](https://makaho.sk8.inrae.fr/)</br>

[MAKAHO](https://makaho.sk8.inrae.fr/) is a [R Shiny](https://shiny.rstudio.com/) website based on [ashes](https://github.com/super-lou/ashes) and [AoTre](https://github.com/vmansanarez/AoTre) packages. Its main objective is to provide an interactive graphical solution to analyze the hydrologic stationarity of French surface flows.
